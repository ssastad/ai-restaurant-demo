# AI Restaurant Demo

Dette prosjektet er en online demo av AI-bestillingssystem for restauranter.

## Struktur
- backend/: Express API
- frontend/: HTML + JS chat-widget
- .env.example: miljøvariabel (OpenAI nøkkel)

## Deploy
1. Backend → Railway
2. Frontend → Vercel
3. Oppdater widget.js med riktig backend-URL
